# jiji-clone-api

An attempt to develop an Angular clone for jiji Marketplace

## This the REST API Web Service for the jiji clone application. Technologies used:

- Typescript
- Express
- Node
- MongoDB
- SendGrid
- Twilio
- Github Actions
- Jest
- Eslint
- Wanton
